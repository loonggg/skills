# 设计品味准则

## 通用原则（所有风格）

### 排版
- **标题**：衬线体（Noto Serif SC），72px+（漫画）/ 96px（杂志），字间距紧凑
- **正文**：衬线体，32-36px，行高 1.6+
- **标注**：无衬线（DM Sans），20-28px
- **字号对比**：最大与最小 ≥ 6:1
- **禁止 Inter 字体**

### 布局
- **禁止三等分卡片**
- **禁止居中 Hero**
- **区块大小必须有对比**
- **间距不均匀**：有疏密交替

### AI 生成禁忌
- 禁止外发光（box-shadow 光晕）
- 禁止过饱和色
- 禁止通用人名和假数据
- 禁止 AI 文案腔（赋能/无缝/释放/下一代）
- 禁止等分、对称、均匀间距
- 禁止 Unsplash 链接

## 风格一：墨图（黑白漫画）

### 色彩

| 变量 | 值 | 用途 |
|------|------|------|
| `--bg` | `#FAFAFA` | 白底 |
| `--ink` | `#1a1a1a` | 纯黑主色 |
| `--ink-mid` | `#555` | 次要文字 |
| `--ink-light` | `#999` | 标注 |
| `--white` | `#FAFAFA` | 白 |
| `--accent` | `#1a1a1a` | 无独立强调色，用反白 |
| `--tone` | `#E0E0E0` | 灰，辅助色块 |

**强调方式**：纯黑反白（白字黑底），不用彩色。

**禁止**：任何彩色、纯黑 `#000000`、大面积灰色填充。

## 风格二：活力图（暖调漫画）

### 色彩

| 变量 | 值 | 用途 |
|------|------|------|
| `--bg` | `#FBF8F3` | 米白底 |
| `--ink` | `#1E3A5F` | 深海蓝主色 |
| `--ink-mid` | `#5A7A9E` | 蓝灰，次要文字 |
| `--ink-light` | `#9BB5CC` | 浅蓝，标注 |
| `--white` | `#FFFFFF` | 纯白 |
| `--accent` | `#E8613C` | 珊瑚橘，弹点强调 |
| `--tone` | `#F0E6DB` | 暖灰 |

**强调方式**：深蓝反白 + 珊瑚橘弹点 2-3 处。

**禁止**：纯黑 `#000000`、大面积珊瑚橘填充。

## 风格四：杂志图

### 色彩

| 变量 | 值 | 用途 |
|------|------|------|
| `--bg` | `#FBF8F2` | 米白底 |
| `--ink` | `#2C2419` | 深棕主色 |
| `--ink-mid` | `#6B5D4D` | 棕灰 |
| `--ink-light` | `#A89880` | 浅棕标注 |
| `--accent` | `#C75B39` | 赤陶橘强调 |
| `--highlight-blue` | `#3D7EA6` | 静谧蓝高亮 |
| `--highlight-green` | `#4A8C5C` | 橄榄绿高亮 |
| `--highlight-red` | `#C0392B` | 砖红高亮 |
| `--dark` | `#1E2A38` | 深蓝黑（结论区） |
| `--tone` | `#EDE5D8` | 暖灰 |
| `--white` | `#FFFFFF` | 纯白 |

**强调方式**：深棕反白 + 赤陶橘弹点。高亮色按内容情感选一种。

**特殊规则**：
- 标题 96px，左对齐
- 直角边框，无圆角、无阴影
- 区块间用 3px 实线分隔
- Font Awesome 图标可用
- 宽度 1400px
