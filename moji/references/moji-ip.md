# 墨迹 IP 规范

## 墨迹是谁

墨迹是一个黑色实心的小生物，参与系统运作的荒诞工人。

## 外形

- 身体：黑色实心不规则椭圆（`#1a1a1a`）
- 眼睛：两个白色小圆点，无表情
- 腿：两条细线
- 轮廓：略不规则，不是完美几何形
- 表情：面无表情 / 呆滞 / 漠然

## 墨迹的角色

墨迹**不是**：
- 吉祥物
- 贴纸
- 可爱装饰
- 站在角落的旁观者

墨迹**是**：
- 参与系统运作的荒诞工人
- 执行核心动作的行为主体
- 隐喻中的"工人"

## 常见工作

- 搬运材料（抱着箱子、顶着管道）
- 拉扯线条（拽着流程线）
- 被卡在断点（卡在管道口）
- 操作杠杆（按按钮、拉闸）
- 变成漏斗（自己就是容器）
- 切割、盖章、引路
- 举警示牌
- 从洞里伸手出来
- 搬砖、搭桥、分拣、记录

## 判定标准

**如果移除墨迹，隐喻不变，说明墨迹没用对。** 必须重写，让墨迹成为动作的执行者。

## 绝对禁止

- 过于可爱的吉祥物形象
- 儿童卡通风格
- 复杂服装或装饰
- 表情符号 / 闪亮大眼
- 墨迹只是站在角落旁观

## SVG 绘制指南

墨迹用 SVG 绘制，基本结构：

```svg
<!-- 身体：不规则椭圆 -->
<ellipse cx="..." cy="..." rx="20" ry="25" fill="#1a1a1a"
  transform="rotate(-5, cx, cy)" />
<!-- 眼睛：白色小圆点 -->
<circle cx="..." cy="..." r="3" fill="white" />
<circle cx="..." cy="..." r="3" fill="white" />
<!-- 腿：两条细线 -->
<line x1="..." y1="..." x2="..." y2="..." stroke="#1a1a1a" stroke-width="2" stroke-linecap="round" />
<line x1="..." y1="..." x2="..." y2="..." stroke="#1a1a1a" stroke-width="2" stroke-linecap="round" />
```

根据动作不同，添加手臂、工具、姿态变化。身体可以变形（拉长、压扁）以表达动作。
