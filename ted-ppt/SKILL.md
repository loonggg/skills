## 叙事结构设计（必须严格遵循）

### 第1页：开场震撼页
**目标：** 前30秒抓住注意力，制造认知缺口
- **内容选择**（四选一）：
  - 令人震惊的数据对比（"每6秒就有..."）
  - 反常识的观点陈述
  - 引发好奇的问题（"如果我告诉你..."）
  - 个人故事的关键画面（"三年前的那个夜晚..."）
- **视觉要求**：一个超大核心元素 + 极简背景，营造悬念感

### 第2页：为什么这很重要
**目标：** 建立情感连接，让观众意识到"这与我有关"
- 用2-3句话说明问题的普遍性或紧迫性
- 展示问题影响的人群规模或范围
- **视觉策略**：使用人物剪影、地图、或群体图标营造共鸣

### 第3-4页：现状困境
**目标：** 描绘"我们现在在哪里"，设定转折的必要性
- 用具体案例或数据展示当前问题
- 突出传统方法的局限性
- **视觉节奏**：从宽泛到具体，从抽象到可感知

### 第5页：转折点/个人顿悟
**目标：** TED 标志性的"啊哈时刻"
- 讲述发现新视角的瞬间（"直到我意识到..."）
- 展示思维的转变或突破
- **视觉设计**：用光线、箭头、或视角变化象征认知跃迁

### 第6-8页：核心观点展开
**目标：** 深入解释"一个核心idea"（18分钟演讲的黄金法则）
- **第6页**：核心观点的清晰陈述（一句话精华）
- **第7页**：支撑观点的3个关键证据/原理
- **第8页**：用类比或隐喻让抽象概念具象化
  - 示例："这就像..." / "想象一个..."
  - **视觉要求**：插图式隐喻，避免抽象图表

### 第9-10页：真实案例/故事
**目标：** 让观点"活起来"，建立可信度
- 讲述1-2个具体的实践案例
- 包含人物、冲突、转变的完整叙事弧
- **情感设计**：展现真实的挣扎与突破，避免完美主义

### 第11页：潜在质疑回应
**目标：** 预判观众的"但是..."，增强说服力
- 坦诚承认局限性或挑战
- 用数据或逻辑化解顾虑
- **语气**：谦逊而自信，展现思考深度

### 第12-13页：更宏大的意义
**目标：** 从个案上升到普世价值
- 连接到更大的社会议题、人性洞察或未来愿景
- 回答"So What?"——为什么这超越了具体话题本身
- **视觉升华**：使用广阔的场景、星空、地平线等意象

### 第14页：行动召唤
**目标：** 赋能观众，让每个人都能参与改变
- **三层设计**：
  - 你今天就能做的一件小事
  - 你这周可以尝试的挑战
  - 长期可以追求的方向
- **语言**：使用第二人称"你可以..."，而非说教式的"应该"

### 第15页：诗意结尾
**目标：** 留下余韵，制造记忆锚点
- 回扣开场的问题/故事，形成叙事闭环
- 用一句金句或意象定格（可引用或原创）
- **视觉**：极简画面 + 核心金句，营造沉思氛围

## 视觉语言规范（TED 舞台美学）

### 配色哲学
- **主色调**：
  - 背景：深邃的 #1A1A1A（TED 舞台黑）或温暖的 #F5F1ED（知识分享感）
  - 文字：高对比度的纯白 #FFFFFF 或墨黑 #0A0A0A
- **强调色系统**（代表不同情感层次）：
  - **好奇/启发**：TED 红 #E62B1E（仅用于核心观点页）
  - **数据/理性**：科技蓝 #4A90E2
  - **人文/温暖**：琥珀金 #F5A623
- **使用规则**：
  - 每页最多使用1种强调色
  - 强调色占比不超过画面的15%
  - 渐变仅限单色透明度变化（100%-0%）

### 排版铁律

#### 标题层级
- **超大标题**（开场/核心观点）：
  - 字号：屏幕高度的 12-18%
  - 字体：粗黑体/无衬线粗体
  - 最多7个字，去掉所有"的、了、着"
  - 示例：❌"关于创新思维的重要性" ✅"重新定义创新"

- **常规标题**：
  - 字号：屏幕高度的 6-8%
  - 留白：上下至少保持标题高度的 1.5 倍空间

- **正文**：
  - 字号：屏幕高度的 2.5-3.5%
  - 行距：1.8-2.0 倍
  - 每页不超过 30 个字（约 2 行）

#### 数字的力量
- 关键数据必须：
  - 字号放大到标题的 150-200%
  - 单独成行，周围大量留白
  - 配微型单位说明（字号为数字的 1/4）
  - 示例：**73%** <sub style="font-size:0.25em">的人会...</sub>

### 视觉焦点策略

#### 每页组成公式
```
1个视觉锚点（占画面40-60%）
+ 1个核心信息（文字或数据）
+ 呼吸空间（至少35%留白）
= 1张幻灯片
```

#### 图像使用原则
- **摄影图**：
  - 仅用于故事页（案例/情感连接）
  - 必须是真实场景，禁止商业库存图
  - 应用黑白滤镜或单色叠加保持克制感
  
- **图标/插图**：
  - 线性风格，笔触粗细统一（2-3px）
  - 尺寸两极化：要么占屏幕60%（作为主视觉），要么小于8%（辅助说明）
  - 示例场景：用手绘箭头表示流程，用简笔人物表示角色

- **数据可视化**：
  - 禁止3D效果和装饰性元素
  - 最多展示3-4个数据点
  - 颜色仅用于区分类别，不传达数值
  - 优先使用：条形图（对比）、折线图（趋势）、单一饼图（占比）

#### 隐喻可视化技巧
- 将抽象概念转化为日常物品：
  - "复杂系统" → 齿轮组
  - "连接" → 神经网络/星座连线
  - "成长" → 树木/阶梯剪影
- 使用负空间创造双重含义的图形

### 布局网格系统

#### 页面结构
- **全屏沉浸式**：内容垂直+水平居中，四周边距至少10%
- **不对称平衡**：
  - 视觉重心可偏离中心，但需用留白或微元素平衡
  - 文字靠左/右时，另一侧放置呼吸性图形元素

#### 动线设计
- 从左到右、从大到小的阅读习惯
- 关键信息放在画面上1/3"黄金视线区"
- 次要信息沉到底部，字号缩小至正文的70%

## 交互与动效（增强沉浸感）

### 导航系统
- **进度指示**：底部圆点（●○○○），当前页放大并变色
- **快捷键**：
  - 空格/右箭头：下一页
  - 左箭头：上一页
  - 数字键：直达对应页
- **手势**（移动端）：左右滑动切页

### 页面转场
- **默认动效**：0.6秒缓动淡入淡出（ease-in-out）
- **特殊时刻**：
  - 开场到第2页：向上推入（push up）
  - 转折点页：缩放淡入（scale fade）
  - 结尾页：长淡入（1.2秒）营造余韵

### 元素进场
- 标题：从透明度0到100%，伴随轻微上移（20px）
- 数据：数字从0跳动到目标值（1秒动画）
- 图像：从模糊到清晰（blur 10px → 0px）

## 内容创作检查清单

### 叙事完整性
- [ ] 是否有明确的"一个核心观点"可以用一句话概括？
- [ ] 开场是否在30秒内制造了认知缺口？
- [ ] 是否包含至少1个真实故事（而非纯理论）？
- [ ] 是否有"啊哈时刻"的设计？
- [ ] 结尾是否给出了具体可行的行动建议？

### 情感曲线
- [ ] 是否设计了情绪起伏（好奇→惊讶→思考→希望）？
- [ ] 是否展示了脆弱性或失败经历以建立信任？
- [ ] 语气是否为对话式而非说教式？

### 视觉一致性
- [ ] 每页是否只有1个视觉焦点？
- [ ] 留白是否达到30%以上？
- [ ] 强调色是否仅用于核心信息？
- [ ] 字体家族是否统一（最多2种字体）？

### 18分钟黄金法则
- [ ] 总页数控制在15页以内（平均每页1分钟讲解）
- [ ] 每页信息量是否能在5秒内被视觉捕捉？
- [ ] 是否删除了所有非必要的修饰词和冗余信息？

## 最终输出要求

### 技术规格
- 响应式设计，支持 1920×1080 / 1366×768 / iPad / 手机横屏
- 使用现代 Web 技术（HTML5 + CSS Grid/Flexbox）
- 字体回退方案：中文（思源黑体/苹方 → 黑体），英文（Inter/Helvetica → Arial）
- 所有动画使用 CSS transitions/animations，避免 JavaScript 性能损耗

### 可访问性
- 文字与背景对比度至少 7:1（WCAG AAA 标准）
- 图像必须有 alt 描述（即使是装饰性图形也要标注"装饰性"）
- 键盘导航完整支持

### 交付物
1. 完整的 HTML 单文件（内联 CSS）
2. 演讲者备注（每页的演讲要点，单独 markdown 文件）
3. 高分辨率导出的静态图片（可选，用于社交分享）

## 创作心法（Meta 层面的提醒）

> **记住 TED 的核心**：不是展示你知道多少，而是点燃观众心中"我也想了解更多"的火焰。

- 每一页都问自己：如果观众此刻走神看手机，我失去了什么？如果答案是"没什么"，删掉这页。
- 数据是为了服务故事，不是相反。先找到故事，再用数据强化冲击力。
- 最好的设计是隐形的——观众应该记住观点，而非"这个 PPT 真炫酷"。
- 18分钟的限制是解放而非束缚，它逼迫你找到思想的钻石核心。

**最终目标**：当观众走出会议室，他们可能忘记你的名字、忘记你的幻灯片，但他们会记住那个改变他们看世界方式的idea。
